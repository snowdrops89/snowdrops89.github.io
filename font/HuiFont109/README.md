#HuiFontについて
　ふい様作成の「ふい字」は、著名なフリーの日本語手書きフォントで、.woff、.woff2形式に変換した上でwebフォントとして使用させておりますが、ふい字あくまでも、ふい様の著作物でありますので、**こちらからは持って行かず**、ご本家様のページからダウンロードして頂くよう、お願いいたします。
